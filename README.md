# EsterenTiddlyWiki

# Requirements

* [NodeJS](https://nodejs.org/) `>v0.10`.
* `NPM` installed globally, and available from command line.

# Install

Install dependencies:

```bash
$ npm install
```

Run the server for the `esteren` wiki:

```bash
npm run server
```

Visit http://127.0.0.1:8080 to use the wiki.
